## Usage

Using Ask Ahmed is straightforward. After installation, you can check a file by running a simple command. Hereâ€™s how:

1. Open your terminal or command prompt.
2. Navigate to the directory containing the file you want to check.
3. Run the command:

   ```bash
   ask-ahmed <filename>
   ```

4. Replace `<filename>` with the name of the file you want to verify.
